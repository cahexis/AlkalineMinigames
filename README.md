# AlkalineMinigames
Project made for resume. smiley face. <br>

This project is a little discord bot that allows you to flip a coin (speaking head emoji for heads, and cat emoji for tails). <br/>
It uses the discord.py library/api wrapper for functionalit, the gif used is not owned by me and credit goes to respective creator. <br/>
If you decide to use this, you will need to setup the bot's permissions and its own token. <br/>

I definitely could improve this in multiple ways, one of them being a balance for each user. I had this at the start before I removed it as I wanted to simplify it a bit more.
